# news_video_maker 開発履歴

海外技術ニュースを日本語ナレーション付きのYouTube Shortsに自動変換・投稿するツールの開発記録。

## 技術スタック概要

| 分類 | 技術 |
|---|---|
| 言語 | Python 3.12 |
| パッケージ管理 | uv |
| ニュース取得 | feedparser + httpx |
| 論文取得 | arXiv API + HuggingFace Daily Papers API |
| LLM処理 | Claude Code（カスタムコマンド） |
| オーケストレーター | Claude Agent SDK (Python) |
| TTS | VOICEVOX（ローカル実行） |
| 映像生成 | Playwright + HTML/CSS + moviepy |
| 背景画像 | Stable Diffusion（SD 1.5 → DreamShaper 8）|
| 背景動画 | AnimateDiff v3（オプション） |
| YouTube投稿 | google-api-python-client (OAuth 2.0) |
